Component Reference

Alle herbruikbare componenten voor het portfolio — met live preview, klasse-overzicht en gebruiksinstructies. Geen component heeft inline styles. Alles staat in styles.css.

Hergebruik Navbar Hero Project Card Footer Gedeeld

Hoe hergebruik je componenten?

Strategie
Navbar & Footer zijn Web Components. Ze leven in js/components.js als <site-nav> en <site-footer>. Je schrijft ze één keer op elke pagina als één HTML-regel. Wil je de tekst in alle footers aanpassen? Dan bewerk je alleen js/components.js — niet elke pagina apart.
Hero en Project Card zijn HTML-patronen. Ze hebben geen JS nodig. Kopieer het HTML-blok, pas de content aan, klassen zijn altijd hetzelfde. Voeg een nieuwe case toe? Kopieer één .project-card blok en vul in.
Component Techniek Bewerk je hier Aanpassen per pagina via
<site-nav> Web Component js/components.js page="…" attribuut
<site-footer> Web Component js/components.js root="…" attribuut
Hero section HTML patroon De HTML-pagina zelf Tekst en links direct aanpassen
Project Card HTML patroon De HTML-pagina zelf card-color-1/2/3/4 voor kleur
Nieuwe pagina toevoegen in 3 stappen:
1. Kopieer about.html als startpunt.
2. Verander page="about" naar de juiste waarde in <site-nav>.
3. Voeg de nieuwe link toe in het NAV_LINKS array in js/components.js. Alle bestaande pagina's krijgen de link automatisch.

04 — Footer

<site-footer> Web Component
© 2024 Annelies Le Page
LinkedIn Dribbble
Gebruik in HTML:

Root pagina:
<site-footer root="./"></site-footer>

Sub-pagina:
<site-footer root="../"></site-footer>

Het jaar wordt automatisch ingevuld via new Date().getFullYear() — je hoeft dit nooit handmatig bij te werken.
AttribuutWaardenEffect
